# Millionaires-Banking-Application-w-GUI-JAVA-SWING-MYSQL
Java-based GUI banking app that enables user account creation, login, and transactions (deposit/withdraw). Integrates with MySQL to store and manage account data securely. Uses prepared statements for safe balance updates and ensures accurate user authentication and data integrity.

# Banking Application

## Overview
This Banking Application is a Java desktop project that simulates basic banking operations through a graphical user interface. Users can create accounts, log in, deposit funds, withdraw funds, and view account balances. The application is connected to a MySQL database to store and manage account information.

## Features
- Account creation for new users
- Secure login for existing users
- Deposit and withdrawal functions
- Real-time balance display
- Database storage with MySQL
- Input validation and error handling

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC

## Purpose
The purpose of this project is to demonstrate core programming concepts such as object-oriented design, GUI development, database connectivity, and transaction handling in a real-world style banking system.

## What I Learned
Through this project, I gained experience with:
- Building Java GUI applications using Swing
- Connecting Java applications to MySQL with JDBC
- Writing SQL queries for storing and updating user data
- Handling exceptions and validating user input
- Designing a simple but functional banking workflow
